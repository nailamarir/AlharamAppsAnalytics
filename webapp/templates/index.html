Pipeline Documentation

Learn how each preprocessing step transforms your data

Username Cleaning

Cleans and normalizes usernames from app reviews.

  • Arabizi Conversion: Mo7amed → Mohamed, 3li → Ali
  • Trailing Numbers: Hasan855 → Hasan
  • Symbol Removal: Removes emojis, punctuation
  • Anonymous Handling: Empty/invalid names → "Anonymous"

Language Detection

Identifies the language of each review text.

  • Arabic: Reviews in Arabic script
  • English: Reviews in Latin script
  • Mixed: Reviews with both scripts
  • Unknown: Cannot determine language

Device Type Mapping

Maps platform to device type for analysis.

  • iOS: App Store reviews
  • Android: Google Play reviews
  • Other: Unknown platforms

App Name Normalization

Standardizes application names to merge duplicates.

  • Arabic Variations: توكلنا → tawakkalna
  • Spelling Fixes: حافلات مكه → حافلات مكة
  • Merged Names: نسك → Nusuk نسك

Service Classification

Categorizes apps by service type.

  • Health: صحتي, أسعفني
  • Reservation: حافلات مكة, قطار الحرمين
  • Government: توكلنا, نسك
  • Religious: مكتشف القبله, مصحف الحرمين

Period Tagging

Tags reviews with Islamic calendar events and holidays.

  • Hajj Season: 1-15 Dhul Hijjah
  • Ramadan: Month 9 in Hijri calendar
  • Eid al-Fitr: 1-3 Shawwal
  • Eid al-Adha: 10-13 Dhul Hijjah
  • School Summer: KSA Ministry dates
